Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The secion element doesn't necessarily add anything visible to the page until other elements are added.
  • There should only be one H1 tag per page.
  • The other numbered heading tags should go in numerical order.

CSS

  • CSS is the language we use to format a webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A box-shadow property adds a box that is described by x first and then y offsets from the element plus blur and radius and color.

Git

  • Git is the terminal interface I use on my local computer to hold and push/pull code to/from the cloud repo GitHub.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-hame: creates a new branch and switches to it.
